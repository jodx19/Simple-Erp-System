# Simple ERP System

A full-stack ERP web application built with Angular and ASP.NET Core for managing business operations.

## Features

- Dashboard analytics
- Orders management
- Customers (CRM)
- Suppliers (SRM)
- POS system
- Inventory management
- Activity trail
- Settings & team management

## Tech Stack

Frontend
- Angular
- SCSS
- Angular Material
- Chart.js

Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
